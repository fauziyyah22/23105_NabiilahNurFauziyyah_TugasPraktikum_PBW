Tentang Saya

Saya Nabiilah Nur Fauziyyah, mahasiswa semester 4 jurusan Informatika di Universitas Singaperbangsa Karawang. Saat ini, saya sedang mendalami pemrograman web dengan HTML, CSS, dan JavaScript.

Proyek

Tabel Data Penjualan

Tabel berbasis web untuk mencatat dan menampilkan data penjualan bulanan. Menggunakan HTML dan Tailwind CSS, tabel ini memungkinkan pengguna untuk menambahkan, melihat, dan mengelola data penjualan dengan tampilan yang interaktif dan mudah digunakan. Selain itu, sistem ini mendukung penghitungan otomatis total penjualan serta tampilan tabel yang responsif.

Portfolio
